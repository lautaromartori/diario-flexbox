Portada de diario digital responsivo creado con HTML y CSS utilizando Flexbox y Mobile First. Incluye noticias principales con imágenes, noticias secundarias, menú de navegación adaptativo con ícono hamburguesa para móviles, y publicidades.
El diseño se adapta perfectamente a cualquier dispositivo mediante media queries.
